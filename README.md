# Mohamed Amine Belachhab

Personal academic and creative website for Mohamed Amine Belachhab, an independent student researcher from Fes, Morocco.

The site presents my research papers, supporting archives, creative/public work, and contact links in a simple static format. It is built with plain HTML and CSS and hosted through GitHub Pages.

## Website

Live site:

```txt
https://aminb-el.github.io/me/
```

## Main sections

* **About** — short profile and overview of my research and creative work.
* **Research** — public PDFs of working papers, empirical studies, methodology documents, and links to the GitHub research archive.
* **Creative Work** — photography editions, calligrams, visual booklets, public projects, and teaching-related work.
* **Contact** — email, GitHub, ORCID, PhilPapers, Zenodo records, YouTube, and other public links.

## Research archive

The main research archive is kept separately here:

```txt
https://github.com/aminb-el/research
```

It contains source files, supporting materials, logs, code, datasets, revision materials, and reproducibility files where available.
